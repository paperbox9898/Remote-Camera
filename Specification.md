# 개발 예정

1. Tailscale 사용
   - 임시 터널 주소가 계속 바뀌는 문제를 해결한다.
   - 휴대폰 앱에서 집컴 서버를 고정 주소로 접속할 수 있게 한다.
2. 앱에 알람 피드백
   - OK/NG 결과를 더 명확하게 표시한다.
   - NG 발생 시 소리, 진동, 화면 강조를 추가한다.
3. 앱 동영상 촬영
   - 사진 1장 판정에서 짧은 동영상 업로드 판정으로 확장한다.
   - 필요 시 프레임 단위로 샘플링해 서버에서 판정한다.
4. 그래픽 카드 사용
   - 집컴 GPU 사용 가능 여부를 확인한다.
   - TFLite/YOLO 실행 방식을 GPU 또는 더 빠른 런타임으로 개선한다.
5. Claude API로 분석
   - 감지 결과 이미지와 이력을 요약하거나 위험 상황 설명을 생성한다.
   - 오탐/미탐 사례를 사람이 검토하기 쉽게 정리한다.
6. 자동 학습 및 개선
   - 오탐/미탐 이미지를 모아 재학습 데이터셋으로 관리한다.
   - 운영 데이터 기반으로 모델 개선 루프를 만든다.
